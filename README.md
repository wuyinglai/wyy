# 余烬商队

项目定位：
商队远行 + 卡牌战斗 + 跑商经营 + 城市重建

当前阶段：
Phase 1：项目骨架和初始状态展示

## 运行方式

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 类型检查

```bash
npm run typecheck
```

## 测试

```bash
npm run test
```
